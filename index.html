SwitchButton

A simple set of toggle buttons for faster prototyping in code

SwitchButton uses unicode symbols and the jQuery library to toggle an active class when clicked. It helps interaction designers & front-end developers to quickly prototype toggle button interactions.

SwitchButton is based on unicode symbols and can be used with icon fonts like Symbolset, Foundation Icons (used in this demo), Entypo, or Font Awesome.
